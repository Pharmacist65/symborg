# Contributing

SYMBORG is currently a concept and prototype-direction repository. Contributions are most useful when they make the project more precise, testable, or ethically grounded.

## Useful Contributions

- Critique of the architecture
- Better wording for overbroad claims
- Research sources
- HCI evaluation metrics
- Accessibility and assistive communication feedback
- Prototype ideas
- Risk analysis
- Interface design constraints

## Contribution Style

Please keep feedback specific.

Good:

```text
The architecture should distinguish factual retrieval from personal memory retrieval because the privacy requirements differ.
```

Less useful:

```text
This will change everything.
```

## Boundaries

Do not submit claims that imply:

- Medical efficacy
- Mind reading
- Company affiliation
- Approved clinical use
- Guaranteed cognitive enhancement
