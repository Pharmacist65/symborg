# Visual Assets

These images define the initial visual direction for SYMBORG.

The intended style is:

- Premium
- Human-centered
- Medical-grade
- Calm
- Non-dystopian
- Non-cyberpunk
- No readable UI text
- No logo dependency

## Files

- `01_symborg_hero_vision.png`: Main project hero
- `02_symborg_product_family.png`: Wearable device family
- `03_contextual_whisper_scene.png`: Social-context use case
- `04_brain_frontend_backend.png`: Cognition frontend/backend metaphor
- `05_cognitive_packet_engine.png`: Knowledge compression concept
- `06_neural_load_aware.png`: Attention and load-aware delivery
- `07_private_memory_vault.png`: Local-first private memory module
- `08_haptic_ring_control.png`: Silent haptic control language
- `09_assistive_medical_use.png`: Assistive communication direction
- `10_future_use_cases_triptych.png`: Learning, meeting, and social intelligence use cases
- `contact_sheet_symborg_visuals.png`: Overview sheet
- `prompts.md`: Source prompt direction for regenerating or extending the visual system

## Usage Note

These assets are concept visuals. They should not be used to imply an existing manufactured product, clinical validation, or affiliation with any company.
